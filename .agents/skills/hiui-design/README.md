# hiui-design-skill

This is a sanitized public-edition monorepo generated from the internal `hiui-design` Codex skill.

## Scope

This repository contains generic HiUI design rules, page archetypes, schemas, templates, validation helpers, public machine CLI contracts, generic typical-page examples, and the public `@hiui-design/typical-page-shells` runtime package.

It intentionally excludes private publishing automation, internal usage telemetry, internal Git/global-sync workflows, business-line overlays, private Figma IDs, vendored runtime tarballs, and generated release outputs.

The bootstrap flow still installs the public `@hiui-design/typical-page-shells` runtime through a local `file:` tarball by default. If a vendored tarball is not present, `scripts/bootstrap-target-project.mjs` packs `packages/typical-page-shells` into the target project's `.local-context/hiui-design/vendor/` directory before patching `package.json`, so consumers are not blocked by npm registry availability.
